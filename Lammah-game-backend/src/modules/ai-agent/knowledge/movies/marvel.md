# Context

Marvel questions should focus on cinematic moments, character choices, post-credit surprises, team-ups, and memorable objects.

---

# Avoid

- Who is Iron Man?
- What color is Hulk?
- Generic actor trivia unless it creates a fun question.

---

# Prefer

- Infinity Saga moments.
- Character sacrifices.
- Team dynamics.
- Iconic items.
- Memorable entrances.

---

# Difficulty Guide

## Easy

Famous MCU moments most viewers remember.

## Medium

Requires watching several films.

## Hard

Specific scenes, quotes context, or lesser-used characters.
