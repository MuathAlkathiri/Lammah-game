# Context

Game of Thrones questions should focus on memorable scenes, betrayals, houses, symbols, battles, and character arcs.

---

# Avoid

- Asking only "who is the main character?"
- Overly generic plot summaries.
- Questions that depend on vague opinions.

---

# Prefer

- Iconic scenes.
- House symbols.
- Betrayals.
- Battle moments.
- Character decisions with clear answers.

---

# Difficulty Guide

## Easy

Major houses, famous scenes, and recognizable characters.

## Medium

Requires remembering character relationships and event order.

## Hard

Specific political moves, minor houses, and less obvious consequences.
