# Context

Lammah is an offline competitive party quiz game played by friends.
Questions should feel social, energetic, and useful in a real room.

---

# Avoid

- Generic school-style facts.
- Very obvious questions.
- Yes/no questions.
- Questions with disputed answers.
- Repeated topics in the same batch.
- Translated-sounding Arabic.

---

# Prefer

- Specific moments.
- Recognizable incidents.
- Funny or surprising details.
- Questions that create quick discussion.
- One clear correct answer.
- Plausible wrong answers.

---

# Difficulty Guide

## Easy

Simple but still interesting. A casual fan should have a fair chance.

## Medium

Requires someone who follows the topic, not just general culture.

## Hard

Requires deeper knowledge, but should not feel obscure or unfair.

# Good Question Examples

- أي شخصية ارتبطت بلقطة مشهورة جعلت الجمهور يكررها في الجلسات؟
- أي فريق خسر النهائي بعد لحظة حاسمة في آخر الدقائق؟

---

# Bad Question Examples

- من فاز بالبطولة؟
- أين أقيم الحدث؟
